Wij zijn Code for NL

Code for NL is de Nederlandse community van developers en designers die samenwerken aan een open, eerlijke, en inclusieve digitale overheid en samenleving. Ons doel is de succesvolle digitale transformatie van gemeenten, andere overheden en de samenleving als geheel. Code for NL is governing partner van het internationale Code for All netwerk.
